## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-01-01 | Tampering | toBundleDir path construction | mitigate | `assertValidSlug(slug)` as first statement; regex `^[a-z][a-z0-9-]{0,63}$` rejects `../`, `/`, `\`, spaces, shell metacharacters before `path.join` executes. | T-02-01-02 | Tampering | toLaunchdLabel / toPlistPath launchd identifier construction | mitigate | Same regex; additionally, `launchctl` would reject labels with `/` or spaces, but fail-fast at the builder is preferable for diagnostic clarity. |

| T-02-01-04 | Information Disclosure | assertValidSlug error message | accept | Error includes `JSON.stringify(slug)` which echoes the invalid input. This is intentional for developer diagnostics; slugs are not secrets (they're directory names). No PII risk. |
